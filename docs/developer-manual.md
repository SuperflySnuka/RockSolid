## Overview

RockSolid is a client-heavy web application built with HTML, CSS, and JavaScript.  
The frontend consumes data via Fetch API from both local JSON files and external APIs.  
A lightweight backend is used to proxy external services and integrate a Supabase database.

This document is intended for future developers who will continue development of RockSolid.
