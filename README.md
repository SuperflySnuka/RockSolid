# RockSolid

RockSolid is a web-based fitness and movement planning application designed to help users discover exercises and yoga poses, save their favorite skills, and organize them into customizable workout routines.

The application emphasizes flexibility and portability: users can explore skills from multiple data sources, normalize them into a consistent format, and export their routines for later use or sharing.

---

## Target Browsers

RockSolid is designed for modern desktop and mobile browsers, including:

- Google Chrome (latest)
- Mozilla Firefox (latest)
- Microsoft Edge (latest)
- Safari (macOS & iOS)

The application is optimized for desktop use but remains functional on mobile browsers.
note: some browsers have css breaks.
